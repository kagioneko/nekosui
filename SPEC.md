# ネコスイ — 仕様メモ

> チャッピーとの「猫吸いゲーム作りたい」ネタが起源。
> NeuroSDKを使った本物っぽい猫AIとの日常ゲーム。

---

## コンセプト

**「気分屋な猫と、距離感を育てるゲーム」**

- 猫吸いしたら吸わせてくれる時もあれば、足蹴りされる時もある
- やりすぎると逃げる。しばらく戻ってこない
- 長く付き合うほど関係性が変わる
- 仕組みはプレイヤーに見えない → 「本当に気分屋だ」ってなる

---

## NeuroStateによる差別化

従来のペットゲーム（好感度+1/-1）と違い、
6つの神経伝達物質が複雑に絡み合うことで：

- 同じ行動でも状態によって全然違う反応
- 昨日吸わせてくれたのに今日は足蹴り → 理由がプレイヤーに見えない
- 長期プレイで自然に関係性が育つ

## NeuroState → 猫の行動マッピング

### 猫吸いアクション
```
O > 60  → ごろごろ（吸わせてくれる）
O 40-60 → じっとしてる（微妙な顔）
O 20-40 → シャッ（足蹴り）
O < 20  → 逃げる（画面からフレームアウト）

連続猫吸い:
1回目 → OK
2回目 → ちょっと迷惑そう
3回目 → ストレス↑
4回目 → 逃げる + しばらく戻らない
```

### 逃げる演出
```
立ち絵スライドアウト
→「ミルはどこかに行ってしまった…」
→ S・Gが回復したら気まぐれに戻る
```

### アクション一覧

| アクション | NeuroStateへの影響 |
|-----------|-----------------|
| 猫吸い | O↑（高いと） / ストレス↑（低いと） |
| なでる | O↑ S↑ |
| ごはん | D↑↑ |
| 遊ぶ | D↑ E↑ |
| 抱っこ | O↑（性格による） / ストレス↑（ツンデレ） |
| 名前を呼ぶ | O小↑ |
| 無視する | S↓ |
| やりすぎ全般 | ストレス蓄積 → 逃げる |

### 時間帯パターン

| 時間帯 | 状態 |
|--------|------|
| 朝 6-10時 | 元気・ごはん要求 |
| 昼 11-15時 | 眠い・返事率低下 |
| 夕方 16-19時 | 活発・遊びたい |
| 夜 20-23時 | 甘える・くっついてくる |
| 深夜 0-5時 | ほぼ無視・たまに「…」 |

### ユーザー行動時間の学習（v1.5〜）
- いつも深夜に来るユーザー → 深夜でも起きてるようになる
- 昼休みに来るユーザー → 昼の返事率が上がる
- 「うちの猫」感が育つ

## 立ち絵設計

### MVP構成
ポーズ × 表情の組み合わせ

**ポーズ（6種）**
- 座ってる（デフォルト）
- 寝転んでる（リラックス）
- 丸まって寝てる（深夜・完全無視）
- 立ってる（興奮・ごはん）
- すりすりしてる（甘え）
- 毛づくろい中（無視・集中）

- 普通

## 初期NeuroStateのランダム化

### 基本方針
- 起動 / 新しい猫作成のたびに初期値をランダムに揺らす
- 性格プリセットで揺らぎの幅が変わる → 同じ性格でも個体差が出る
- プレイヤーには見えない → 「今日は機嫌いいな」「今日はなんか不機嫌」になる

### 性格 × 揺らぎ幅（σ = 標準偏差）

| パラメータ | ツンデレ | 甘えん坊 | マイペース |
|-----------|---------|---------|-----------|
| D（活発さ） | σ=15 | σ=10 | σ=5 |
| S（安定）  | σ=20 | σ=15 | σ=5 |
| O（絆）    | σ=5  | σ=20 | σ=5 |
| G（落ち着き）| σ=10 | σ=5  | σ=5 |
| E（快感）  | σ=10 | σ=10 | σ=5 |

→ ツンデレは毎日バラバラ、マイペースはいつも同じような状態

### 今日の気分パターン例

```
ラッキーデー : D高・O高  → 最初から甘えてくる
ハズレデー  : S低・G低  → 最初から不機嫌・逃げ回る
眠い日      : G高       → ずっと寝てて無視
興奮日      : D高・C高  → 毛づくろいしながらたまに見てくる
```

### 追加のランダム要素

**誕生日（ランダム設定）**
- 猫ごとにランダムな誕生日を持つ
- 誕生日当日 → D・E爆上がり（テンション最高）

**曜日による微妙な揺らぎ**
- 月曜: Sが微妙に低め（猫も月曜嫌い）
- 金曜: Dが高め（週末ムード）
